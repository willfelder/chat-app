## Installation

There is no need to install anything.

[Visit the website here.](https://willfelder.github.io/chat-app/)

If you prefer:

- Clone repository

```
git clone https://github.com/willfelder/chat-app.git
``` 
- Use your favorite IDE
- Execute npm install
- Execute npm start
